## 프로젝트 소개
#### 전통주 특화 쇼핑몰
와인, 샴페인, 맥주 등 기존에 종류별로 찾아보고 비교해보기 쉬웠던 양주 시장처럼, 
우리나라의 전통주를 개인의 취향에 맞게 추천, 소개하고 판매하는 온라인 플랫폼

#### 술학능력시험 – 주(酒)능
전통주 취향테스트 주능(술능)을 통해 취향에 맞는 술 추천 서비스를 제공하여,
소비자가 전통주에 쉽게 접근할 수 있도록 진입 장벽을 낮춰줍니다
